Generates a set of quasi-\/random points using the Halton sequence in N dimensions.\+This function generates {\ttfamily count} points in the unit hypercube using the Halton sequence, then rescales them to fit within the specified axis-\/aligned bounding box. An optional {\ttfamily seed} is used as a starting index offset (i.\+e., a shift) in the sequence to decorrelate multiple calls.
