{\bfseries{Point\+Sampler}} is a lightweight, header-\/only C++ library for point sampling in arbitrary dimensions. It offers generic and reusable algorithms for\+:


\begin{DoxyItemize}
\item Random point generation
\item Distance-\/based filtering
\item Utility operations on points
\end{DoxyItemize}

The library is template-\/based and dimension-\/independent, making it suitable for 2D, 3D, or higher-\/dimensional spaces.\hypertarget{index_usage_sec}{}\doxysection{\texorpdfstring{Usage Example}{Usage Example}}\label{index_usage_sec}
