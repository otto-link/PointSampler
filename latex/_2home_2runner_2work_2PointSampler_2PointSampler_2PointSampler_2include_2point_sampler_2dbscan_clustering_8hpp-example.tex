Density-\/\+Based Spatial Clustering of Applications with Noise (DBSCAN).Groups points into clusters based on density\+: a point is a core if it has at least {\ttfamily min\+\_\+pts} neighbors within distance {\ttfamily eps}. Clusters are formed by expanding from core points. Noise points remain unclustered.
