 {\ttfamily Point\+Sampler} is a lightweight C++ header-\/only library for generating and manipulating points in arbitrary dimensions (1D, 2D, 3D, ... N-\/dimensional). It includes utilities for random sampling, jittered grids, importance sampling, density warping, and spatial relaxation.

The library is designed to be generic, extensible, and dimensionality-\/independent via {\ttfamily std\+::array} and C++ templates.\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item ✔️ Random sampling in arbitrary dimensions ~\newline

\item ✔️ Grid-\/based sampling with jitter and stagger ~\newline

\item ✔️ Importance sampling via rejection or resampling
\item ✔️ Poisson disc sampling (uniform and non-\/uniform)
\item ✔️ Point cloud filtering and normalization ~\newline

\item ✔️ Density-\/based spatial distribution
\item ✔️ Relaxation using k-\/nearest neighbors (with {\ttfamily nanoflann}) ~\newline

\item ✔️ Dimension-\/agnostic (supports any {\ttfamily N})
\item ...
\end{DoxyItemize}
